# Icons Directory

이 디렉토리에는 웹사이트의 아이콘들이 저장됩니다.

## 필요한 아이콘 파일들:
- favicon.ico (16x16, 32x32, 48x48)
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png
- apple-touch-icon.png (180x180)

## 생성 방법:
1. 로고 또는 브랜드 아이콘을 준비
2. 온라인 파비콘 생성기 사용 (예: realfavicongenerator.net)
3. 또는 디자인 도구로 각 크기별로 생성

## 참고:
- PWA 지원을 위해 다양한 크기 필요
- maskable 아이콘도 고려
- 배경색은 투명하거나 브랜드 색상 사용
